# [RELEASE] Universal Roblox ESP [V.1.0]

> Stable ESP with player boxes, health bars (green/red), tracers and custom nametags.

## Features

- Player boxes that scale with distance
- Dual-color health bars (green/red)
- Tracers from screen center to players
- Custom nametags (replaces default)
- Original Roblox names automatically hidden
- Distance-based culling

## Supported Executors

Works on all executors: Delta, Synapse, Krnl, ScriptWare, Fluxus, Electron, etc.

## Usage

1. Copy the script
2. Open your executor
3. Paste and execute

## Configuration
Edit `SETTINGS` table to customize colors, distance, thickness, etc.
